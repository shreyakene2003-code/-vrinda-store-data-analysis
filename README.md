 Vrinda Store Data Analysis Project

 Project Overview

This project focuses on analyzing store data to extract meaningful insights related to sales performance, customer behavior, and profit trends. The analysis helps in understanding business growth patterns and supports better decision-making.

---

 Objectives

* Analyze sales and profit data
* Identify top-performing products and categories
* Understand customer purchasing behavior
* Discover trends and patterns in sales

---

 Tools & Technologies Used

* Microsoft Excel
* Data Cleaning & Data Processing
* Pivot Tables
* Data Visualization (Charts & Graphs)

---

 Key Insights

* Identified highest sales-generating products
* Analyzed monthly and yearly sales trends
* Compared profit across different categories
* Found customer buying patterns

---

 Dataset Information

The dataset contains:

* Order details
* Customer information
* Sales and profit data
* Product categories

---

 Features of the Project

* Cleaned and structured raw data
* Created interactive dashboards
* Used pivot tables for analysis
* Visual representation using charts

--- 

Skills Used

- Data Analysis
- Excel
- Data Visualization
- Pivot Tables

 How to Use

1. Download the Excel file
2. Open it in Microsoft Excel
3. Explore different sheets and dashboards
4. Analyze charts and insights

 Learning Outcome
